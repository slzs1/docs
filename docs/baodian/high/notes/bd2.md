# 插件

## Nprogress

在页面跳转时在最上方显示绿色进度条的插件

![1585188513_1_.jpg](https://i.loli.net/2020/03/26/bYlL9HGJR4Tyj28.png)

### 安装

```
yarn add -D @vuepress/plugin-nprogress
```

### 使用

```
module.exports = {
  plugins: ['@vuepress/nprogress']
}
```
